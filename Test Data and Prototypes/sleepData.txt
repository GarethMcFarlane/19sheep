
"Sleeps" is an array of ~30 days.  Each day contains data about that nights sleep.


The "id" represents one day.

"startTime": the time sleep started.
'Duration': duration of sleep in seconds
"Sleepdetails" is an array within each day that shows periods of sleep.  It's hard to tell but it looks like each datetime is recorded when sleep changes e.g. deep sleep to light sleep or light sleep to awake.

1 = Awake
2 = Light sleep
3 = Deep Sleep

For example the first two datetimes "2015-07-11T21:54:49+10:00" and "2015-07-11T22:00:49+10:00" show that the user had light sleep between 9:54 and 10:00 before going into deep sleep.
